# Lila language learning app

No vibecoding.

- Frontend in react ts vite
- Reader component for reading extracts, chat component for LLM chat
- Backend in Python, connect to amazon ddb
- Dict cc downloaded en<->de, use this for good information about words
- Google translate backend option
- LLM for individual word translations
